# Pages Modal Demo

A small React + Vite + TypeScript demo that implements a reusable `PagesModal` component using Material UI (MUI) v5.

Run locally:

```bash
# install
npm install

# dev server
npm run dev
```

Notes:
- The modal matches the Figma screenshot: header with "All pages", a list of pages, and a golden "Done" button.
- The component is in `src/components/PagesModal.tsx`.
